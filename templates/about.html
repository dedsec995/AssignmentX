1) Enter Your Beautiful Name.


2) Select your Primary and Secondary Colors. You can activate your secondary color by writing text between "$" sign. i.e. $Text Here$, so the text between dollar sign will be written in secondary color.


3) If you have your Text file then select yes and if you do not have your text file then select no and a textbox will appear where you have to add whatever you want in your assignment page


4) Set the opacity if your text. It is optional. It is set on normal, but if you want your text to be more dark or light feel free to experiment with the slider

5) Select form the variety of pages available.. Your output will be printed on the one you sleect.


6) If you have your custom fonts then select yes and if not then dont worry we got you covered. Just select no and we will do the work for you.


7) Depending upon your selection above, store all the files in one place and select them all simultaneously. If you want to select your Text file and Font file then just click on select fils and select them and import them together. If you press select files again then your previous selection will be cleared.


8) Now all you need to do is Click on Upload and wait for our program to build your assignment.
